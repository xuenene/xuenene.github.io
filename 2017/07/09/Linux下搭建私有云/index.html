Linux下搭建私有云

发表于 2017-07-09

受气与各大云盘的龟速上传下载速度和略不友好的文件UI，一直想自己搭个私有云盘用于工作学习。无意中看到阿里云搞活动，租了个oss试试水。

租用oss云服务后，通过官方release包 + README即可完成搭建。搭建后可像本地文件系统一样操作，读小文件基本不用缓冲时间，读大文件或写入文件时速度取决于网速，但一般快过各大网盘。

我下载的是ossfs_1.80.2_ubuntu16.04_amd64.deb。

租用oss云服务

笔者租的是的是阿里云服务器，100G ￥9.00/年（搞活动挺划算），其余全部选择默认配置。付款后进入控制台，此时之前未开通过oss服务的应该会提示开通才能使用。申请开通后新建一个Bucket并记下（相当于一个盘符，客户端访问时需要使用这个Bucket），右上角的“获取accesskey”进入后记下ID和secret，后面会用到。在主界面点击进入Bucket，选择“Bucket属性->域名管理”，记下外网域名，后面会用到。

云端操作完成。
